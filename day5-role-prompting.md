# 30-Day Python Learning Roadmap

## Week 1: Foundations (Days 1–7)
**Milestone:** Understand Python syntax, variables, data types, and basic control flow.

| Day | Task |
|---|---|
| 1 | Install Python & set up VS Code. Write your first "Hello World" script. Learn print(), comments, and running .py files. |
| 2 | Variables, data types (int, float, str, bool), type casting. |
| 3 | Strings: slicing, concatenation, f-strings, common string methods. |
| 4 | Numbers & operators: arithmetic, comparison, logical operators. |
| 5 | Conditionals: if/elif/else. Build a simple calculator. |
| 6 | Loops: for and while, range(), break/continue. |
| 7 | **Review + Mini Project:** Number guessing game (uses conditionals, loops, input()). |

**Resources:**
- [Python.org official tutorial](https://docs.python.org/3/tutorial/)

**Milestone:** Build complete, portfolio-ready projects and get comfortable with external libraries.

| Day | Task |

| 28 | **Capstone Project Day 1:** Plan and start a To-Do List app (CLI, with file storage for persistence). |

**Resources:**
- [Requests library docs](https://requests.readthedocs.io/)

## Final Outcome
By Day 30, you'll be able to:
- Write clean, functional Python scripts using core language features
- Work confidently with files, APIs, and basic data (via pandas)
- Understand and apply basic OOP principles
- Have **3 portfolio projects** on GitHub: a terminal game, a CSV-based expense tracker, and a capstone to-do list app
- Be ready to move into a specialization: web dev (Flask/Django), data science (pandas/numpy deeper dive), or automation scripting

**Tips for success:**
- Code every day, even if just 30 minutes — consistency beats marathon sessions
- Type out code yourself rather than copy-pasting from tutorials
- When stuck, try debugging for 15–20 min before looking up the answer
